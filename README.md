# Histopathological Classification of Lung and Colon Cancer CNN
This DL project implements a Convolutional Neural Network used to classify histopathological lung and colon cancer images into 5 classes and interprets the results through a Grad-CAM explainable method.

The project includes:
- image preprocessing
- data preparation / augementation
- CNN architecture
- CNN training
- performance evaluation
- Grad-CAM visualization

Technologies used:
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib
- Scikit-learn

Dataset: https://www.kaggle.com/datasets/javaidahmadwani/lc25000
